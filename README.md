# JengYeh_WEB
A textbook to teach students at JENG YEH elementary school in Kinmen.

## 週次與授課內容

| 週次 | 授課內容                       |
| ---- | ------------------------------ |
| 1    | 網頁語言介紹                   |
| 2    | HTML head & meta                |
| 3    | HTML 文字、標題、段落           |
| 4    | HTML 連結、圖/影片              |
| 5    | HTML 表格、列表                 |
| 6    | HTML 輸入                      |
| 7    | CSS Selector(選擇器)、Border(邊界) |
| 8    | CSS Padding(內距)、Margin(外距) |
| 9    | CSS Position(位置)、Color(顏色) |
| 10   | CSS Animation(動畫)             |
| 11   | CSS Opacity(透明度)、Float(懸浮) |
| 12   | CSS Flex(佈局)                  |
| 13   | 成果發表                       |
